# Terms

## Defined terms

- `Agent Ledger`: the human-readable governance and continuity system for the Hardlink Organizer project workspace
- `OPEN_LOOP`: a discrete unresolved work item tracked in `OPEN_LOOPS.md`
- `Hardlink Organizer`: the Unraid-focused project in this workspace for safe hardlink planning, execution, verification, and future destination-management workflows
- `Source set`: a configured ingress or download root that is scanned for candidate items
- `Destination set`: a configured library root used by the existing link workflow
- `Destination registry`: the planned database-backed registry of managed destination roots that will supplement config-defined destination sets
- `Link history`: the persisted record of prior link executions or dry runs
- `Verification run`: a persisted audit pass that checks whether expected destination files are true hardlinks to the source files
- `Source of truth`: the authoritative place where project state is recorded
