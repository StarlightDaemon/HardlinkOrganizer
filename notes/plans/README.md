# Hardlink Organizer Plan Archive

This folder holds older implementation and release planning documents that are
still useful reference material but no longer need to live at the repository
root.

## Files

- `IMPLEMENTATION_PLAN.md`: original CLI-first implementation plan
- `V1_RELEASE_PLAN.md`: release-shape planning for the hosted web UI direction
- `WEB_APP_IMPLEMENTATION_PLAN.md`: staged plan for the web-app transition

## Working rule

Use these as background or historical planning references. Prefer the current
project state and open-work flow under `agent-ledger/` when deciding what to do
next.
