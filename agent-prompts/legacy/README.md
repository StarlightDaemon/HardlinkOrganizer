# Legacy Prompt Material

This folder keeps older prompt and handoff material that is still worth
tracking but should not clutter the repository root.

## Files

- `CLAUDE_CODE_HANDOFF_PROMPT.md`: early CLI-first repo-agent handoff prompt
- `WEB_APP_CODE_AGENT_PROMPT.md`: early web-app transition handoff prompt
- `SMALL_SLICE_PROMPTS.md`: older prompt index retained for continuity

## Working rule

Prefer the active prompt pack in `../` for new work. Use these files only when
you need historical context on how earlier implementation phases were framed.
